void statisticSourceFile::readFrameAndTypePositionsFromFile ( )
private

Parser the whole file and get the positions where a new POC/type starts. Save this position in p_pocTypeStartList. This is performed in the background

The background task that parsese the file and extracts the exact file positions where a new frame or a new type starts. If the user then later requests this type/POC we can directly jump there and parse the actual information. This way we don't have to scan the whole file which can get very slow for large files.

This function might emit the objectInformationChanged() signal if something went wrong, setting the error message, or if parsing finished successfully.
