343 /* --- frame dropping API ---
344 
345  To limit decoding to a maximum temporal layer (TID), use de265_set_limit_TID().
346  The maximum layer ID in the stream can be queried with de265_get_highest_TID().
347  Note that the maximum layer ID can change throughout the stream.
348 
349  For a fine-grained selection of the frame-rate, use de265_set_framerate_ratio().
350  A percentage of 100% will decode all frames in all temporal layers. A lower percentage
351  will drop approximately as many frames. Note that this only accurate if the frames
352  are distributed evenly among the layers. Otherwise, the mapping is non-linear.
353 
354  The limit_TID has a higher precedence than framerate_ratio. Hence, setting a higher
355  framerate-ratio will decode at limit_TID without dropping.
356 
357  With change_framerate(), the output frame-rate can be increased/decreased to some
358  discrete preferable values. Currently, these are non-dropped decoding at various
359  TID layers.
360 */
