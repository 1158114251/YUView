67 /*
68 kr/kg/kb matrix (Rec. ITU-T H.264 03/2010, p. 379):
69 R = Y + V*(1-Kr)
70 G = Y - U*(1-Kb)*Kb/Kg - V*(1-Kr)*Kr/Kg
71 B = Y + U*(1-Kb)
72 
73 To respect value range of Y in [16:235] and U/V in [16:240], the matrix entries need to be scaled by 255/219 for Y and 255/112 for U/V
74 In this software color conversion is performed with 16bit precision. Thus, further scaling with 2^16 is performed to get all factors as integers.
75 */
