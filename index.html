Welcome to YUView

YUView is a YUV player with an advanced analytic toolset for Linux, Windows and Mac based on Qt. It can show YUV sequences in several YUV formats and supports zooming, easy navigation through the video, side by side view, comparison view, calculation of differences, playlists and more. On this page we will present some of the supported features. Further information can be found in our wiki.

Getting started

In order to run YUView, you can either get a precompiled binary or you can easily compile YUView yourself. To compile YUView, you need to have Qt (> 5.0, 64bit) installed on your PC. Download the YUView source files and open the .pro file with Qt Creator. Now, just build and run YUView. Both MinGW64 and the VS2013 compilers are tested.

Sequence customization and navigation

In its primary use, YUView is a common YUV player with all typical functionalities - just like you know it from other media players: play, stop, pause, which can be accessed by pressing the spacebar for ease of use. Additionally, you can navigate to each frame with a slider, enter the number directly into a box or use the left and right arrow keys. If you want to repeat a sequence or playlist indefinitely, you can choose between "Repeat", "Repeat All" and of course "Repeat Off", just with a single click.

Display your sequence

With this program you can load more than one video and switch between them by using the up and down arrow keys. YUView supports various pixel formats like 4:0:0 8-bit, 4:2:0 Y'CbCr 8-bit planar, 4:2:2 Y'CrCb 8-bit planar, 4:4:4 Y'CbCr 12-bit LE planar, RGB 8-bit and GBR 12-bit planar just to name some of them. We also support different color conversions according to ITU-R Recommendations BT.601, BT.709 and BT.2020. A list of all supported formats can be found in our wiki.

You have small, uncommon or odd frame sizes and/or 4k sequences? No problem! YUView has no built in limit on the frame size. And even if you have a non-standard frame size, you can customize your frame width and height just like the frame rate via several boxes.

YUView also offers the opportunity to inspect a single color component by an easy dropdown menu, if you need to see either luminance only or one of the chrominance components. You can also scale and offset the luminance or chrominance component for further customization. For a closer look, you can zoom into the sequence by selecting the desired area, using the mouse wheel or pressing '+' or '-'.

Overlay statistics

In YUView, you can display additional statistics as an overlay on top of your video. This could be data generated by an encoder/decoder (block size, prediction modes, motion information) or other information generated by a separate process. For this purpose, we define a ".csv" file format that carries information on a block basis for a video sequence. These files can be loaded in YUView and dropped onto a video sequence to show the overlay. A definition of the csv file can be found in our wiki.

See every pixel

If you work with decoders or pixel manipulators, you want to know what happens in each block of your frames. For this, YUView has two features to help you with that: a pixel grid and a zoom box. The pixel grid overlays your frame with horizontal and vertical lines in a user defined spacing. The zoom box makes it even more exact. If activated, you can scroll over each pixel of your frame and the zoom box will show you the coordinates, YUV component value and, if you have some statistics activated for this sequence, the value of the currently selected pixel alongside a small area around this pixel.

Sequence comparison

Compare two videos with different quality levels visually? YUView offers the opportunity to display two videos either in split or in comparison view. In split view, the two videos are shown in full scale side by side. You can navigate and zoom into both videos just like you did before. In comparison view, frames are shown in the same resolution, divided by a vertical bar in the middle. Right of this bar you can see the first video and on the left side, the other video is displayed. This way you can spot differences easily. The vertical bar can be moved to the left or right, so you can compare different frame parts of the video sequences. To activate the sequence comparison, simply select two videos of your playlist and set the checkbox "Split- and Comparison View". Now you can select either choose "Side-By-Side" or "Comparison" in the dropdown menu.

See the difference

If you want to directly see what the exact, per-pixel difference between two sequences is, YUView can generate difference frames. To do that, select two videos of your playlist, right click on them and choose 'Add Difference Sequence'. Now you can visually spot differences of your two sequences directly in a video. Note that you can also change the luminance scale or offset to amplify differences which are hard to spot!

Don't lose time

You started an analysis and want to work on it later? You don't have to look for all sequences and files that you had last time. YUView can create and save your current playlist and all its current settings you chose in a playlist file. Use File->Save Playlist, enter a name and next time you need this playlist, simply reopen it via drag & drop.
